# 高级特性

## 一、切片

**带头不带尾**

对于一个列表L = [1, 2, 3, 4, 5, 6, 7, 8, 9]，我们使用切片来获取特定元素

- 获取第1个到第3个（数组下标从0开始，切片是左闭右开的区间，也就是包含0，不包含3)

  ```python
  L[0:3]
  ```

- 获取第2个到第五个

  ```python
  L[1:5]
  ```

- 取倒数第5个到倒数第2个

  ```python
  L[-5:-1]
  ```
- 取第2个到最后一个

  ```python
  L[3:]
  ```
- 前5个数，每2个取一个

  ```python
  L[:5:2]
  ```
- 所有数，每个两个取一个

  ```python
  L[::2]
  ```
